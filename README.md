# Geolocation Tracker

A Python-based application that fetches a user's geolocation using their public IP address and displays it on an interactive map.

## Features

- Detects user location using IP address
- Displays city, region, and country
- Shows latitude and longitude
- Generates an interactive map using Folium
- Opens the map in the browser automatically

## Technologies Used

- Python
- Requests
- Folium
